¿Que es C++

C++ es un lenguaje de programación diseñado a mediados de los años 1980 por Bjarne Stroustrup. La intención de su creación fue el extender al lenguaje de programación C mecanismos que permiten la manipulación de objetos. En ese sentido, desde el punto de vista de los lenguajes orientados a objetos, el C++ es un lenguaje híbrido.

Usos de C++

Los usos que se le puede dar a C++ son:

Sistemas operativos como Windows, Linux o Mac Os.

Los sistemas conocidos como Microsoft y Linux están todos desarrollados en C++ porque es un lenguaje fuerte y rápido para codificar, lo que lo hace ideal para crear un sistema operativo desde cero.

Videojuegos

Es un lenguaje de programación muy versátil que fue pensado inicialmente para la manipulación de objetos. En cuanto a videojuegos, pese a su antigüedad, sigue siendo muy importante para la creación de todo tipo de títulos, desde juegos de scroll y de luchas hasta aventuras en 3D. C++ permite crear software de bajo nivel del que pueden partir sistemas mucho más complejos. Estos son algunos juegos creados que utilizan C++: Pokemon Go, Brawl Stars, Clash Royale, Among Us.

Usos Extras

Tambien se usa en controladores de dispositivos, HPC, nube, escritorio, incrustado


Desarrollo web con C++

En el contexto del desarrollo web, C++ se usa para construir programas CGI (Common Gateway Interface) que pueden interactuar con cualquier otro sistema externo para intercambiar información. Aunque no es el lenguaje de programación más popular para construir proyectos web, hay al menos tres razones obvias para usarlo:

  • Para hacer frente a cargas ultra altas en las soluciones de gigantes digitales como Google, Facebook, etc.;
  • Para construir software multiplataforma;
  • Para desarrollar interfaces web para dispositivos inteligentes para los que el rendimiento y los recursos informáticos de marcos populares como Django no son suficientes.
